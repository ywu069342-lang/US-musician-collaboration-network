# US-musician-collaboration-network
Data updated in Dec 2023 from musicbrainz and last.fm

listeners1000000US.csv: US artists with more than 1,000,000 listeners in last.fm

adjacency_matrix_popUS100w.csv: Collaboration adjacency matrix of artists with 'pop' tag and  more than 1,000,000 listeners in last.fm
adjacency_matrix_rockUS100w.csv: Collaboration adjacency matrix of artists with 'rock' tag and  more than 1,000,000 listeners in last.fm

adjacency_matrix_popUS200w.csv: Collaboration adjacency matrix of artists with 'pop' tag and  more than 2,000,000 listeners in last.fm
adjacency_matrix_rockUS200w.csv: Collaboration adjacency matrix of artists with 'rock' tag and  more than 2,000,000 listeners in last.fm

artist_tag.csv: Artists' tags in musicbrainz
Popular_name_mbid.csv: Some important mbid-artists name projection
